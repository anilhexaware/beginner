Pipes

Pipes are used to transform the data

Pipes will take data input & convert/transform into a desired format

Pipes are written using the pipe operator (|)

We can apply pipes to any template & in any data inputs

Types of Pipes::

Built in pipes

{{ user1.First | lowercase }}, {{ user1.Middle | uppercase }}

{{ user1.DOB | date: "MMM dd yyyy" }}

{{ user1.salary | currency: "INR" }}

Parameterized pipes

{{ user1.First | lowercase }}, {{ user1.Middle | uppercase }}

{{ user1.DOB | date: "longDate" }}

{{ user1.salary | currency: "INR":"INR":"3.0-0" }}

Chaining pipes

We can add multiple pipes in data input in Chainingpipes

We can pass one or more pipes to a data input

{{ user1.DOB | date: "MMM" | uppercase }}

Custom Pipes

We can generate a new custom pipe using CLI ng g p Highlight

We need to import pipe & pipe transform from @angular/core
